Инструмент проверки в черных списках

Этот инструмент проверяет доменное имя или IP-адрес по нескольким черным спискам.

ссылка на репозиторий

Использование

  • Установите необходимые библиотеки Python:
pip3 install -r requirements.txt
  • Collect the data, checking in blacklists
python3 cmp_report.py build
  • Запустите Flask-приложение:
python3 cmp_report.py &

Затем откройте http://localhost:8002 в браузере.

Вариант с Docker

docker build -t blacklist_report .
docker run -d -p 8002:8002 blacklist_report

Используемые черные списки:

  • zen.spamhaus.org
  • dbl.spamhaus.org
  • multi.uribl.com
  • bl.spamcop.net
  • b.barracudacentral.org
  • hostkarma.junkemailfilter.com
  • dnsbl.sorbs.net
  • dnsbl-1.uceprotect.net
  • ...
