Blacklisting Report Tool

This tool checks a domain name or IP address against multiple blacklists.

Usage

  1. Install required Python libraries:
pip3 install -r requirements.txt
  1. Start the Flask app:
python3 blacklist.py

Then open http://localhost:8002 in your browser.

Docker version

docker build -t blacklist_report .
docker run -d -p 8002:8002 blacklist_report

Blacklist sources used:

  • zen.spamhaus.org
  • dbl.spamhaus.org
  • multi.uribl.com
  • bl.spamcop.net
  • b.barracudacentral.org
  • hostkarma.junkemailfilter.com
  • dnsbl.sorbs.net
  • dnsbl-1.uceprotect.net
