Blacklisting Report Tool

This tool checks a domain name or IP address against multiple blacklists.

Link for repo

Usage

  • Install required Python libraries:
pip3 install -r requirements.txt
  • Collect the data, checking in blacklists
python3 cmp_report.py build
  • Start the Flask app:
python3 cmp_report.py &

Then open http://localhost:8002 in your browser.

Docker version

docker build -t blacklist_report .
docker run -d -p 8002:8002 blacklist_report

Kubernetes, double container pod version

kubectl apply -f k8s/pod_cmp_report.yaml
kubectl port-forward pod/server-report 8002:8002 --address 0.0.0.0

Kubernetes, deployment + service version

kubectl apply -f k8s/deployment_cmp_report.yaml
kubectl apply -f k8s/service_cmp_report.yaml
